# Vagrant-Scripted-Env 

## What's it do? 

Not too much, just a quick and easy way to spin up a vagrant-box after scripting the installation of dependencies. 

## OSX 

1. Clone this repo 
1. ./devops/osx/setup.sh
1. Grab Coffee
1. Enjoy 

## Windows 

1. Clone this repo 
1. .\devops\windows\setup.bat
1. Grab Coffee
1. Enjoy
